# DeviceGuard AI

Predict. Prevent. Protect.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included

- Next.js 15, React, TypeScript, Tailwind CSS, Shadcn-style primitives
- Zustand app state, React Query service layer, Socket.IO client hook
- Recharts dashboards, Framer Motion shell interactions
- Mock API data and typed domain interfaces
- Dashboard, devices, device details, analytics, alerts, maintenance, AI insights, root cause, integrations, executive, notifications, and settings routes
